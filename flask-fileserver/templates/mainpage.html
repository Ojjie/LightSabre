Light Sabre.

A secure, decentralized lugagge tracking service built on the Ethereum platform.

Can be deployed between airlines, airports and ensures no loss of luggage.

The loss of baggage ends here.

Smart contracts for Luggage Blockchain Transactions in Solidity.

Blockchain interface on Ganache. Front end made using React, Bootstrap and Javascript

Light Saber works as both a client side and server side application. Transactions made on the blockchains (addition and removal of blocks between airports) are done on Metamask.io
